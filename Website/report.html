Every other picture is from my own personal collection and are pictures of myself playing basketball

The Rating Star pictures were made by me by pasting together copies of a picture I got from the internet. I tried using one picture embedded several times but could not get them to align properly

I have used JavaScript to do some picture animations. One of them can be found on the main page. Pictures of the daily headlines are displayed on the bottom of the page. When the page loads, the script is activated. A random number is generated and the main picture on the screen is displayed according to that number. All pictures are stored in a vector and the picture changes every few seconds. The change is also emphasized in the bottom images by drawing a blue border on the newly selected picture.

A similar script is used to animate a slideshow of court pictures in the court_presentation section.

The Photos section contains several albums which are created by users and/or staff and each album has a picture gallery which contains thumbnails and descriptions. Clicking a thumbnail will open a new window with the real size picture. also, hovering with the pointer over an image will cause a temporary blue border to pe added.
